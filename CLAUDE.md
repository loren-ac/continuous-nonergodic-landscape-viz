This repo is for exploring the visualization of various aspects and attributes of continuous nonergodic processes.

It will likely be useful to use the simplexity package or some translation of its functionality. Relevant simplexity repos can be found at @/Users/loren/simplexity-repos/. The dev branch is what I usually use but there are other branches that have additional functionality that have not been reviewed for merge into dev.

For references of continuous nonergodic processes, see the simplex-research repo @/Users/loren/simplex-research-repos/nonergodic-continuum. For a reference of a code implementation of regular nonergodic processes see the simplexity repo at @/Users/loren/simplexity-repo/kyle-nonergodic. And, for a reference of the theory behind regular (non-continuous) nonergodic processes see @/Users/loren/simplex-documentation/docs which contains all simplex related documentation outside of code documentation.

For a reference of our current website which gives some ideas about our aesthetic and also has a high-performance demo of our approach that runs in browser see @/Users/loren/simplex-safety-repos/simplex-website.

For a reference of websites whose design, visualizations, ui/ux, and styles that I would like to take inspiration from for this visualizer please see @_reference/. I particularly like the aesthetic of commitmono and the aesthetic, visualizations and ui/ux of integrated-reasoning.

For a design philosophy that I like please specifically see the design philosophy described on the front page of the usgraphics website in @_reference/.
